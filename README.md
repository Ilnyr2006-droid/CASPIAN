# CASPIAN — «динамический» контент без сервера

Этот вариант оставляет ваш дизайн (HTML/CSS) как есть, но делает **товары / новости / проекты** динамическими:
- данные лежат в `data/*.json`
- страницы рендерят данные через JS (`js/dynamic-*.js`)
- язык подтягивается из `localStorage.caspian_lang` так же, как и в вашем `lang.js`

## Что менять, чтобы добавлять контент
- Товары: `data/products.json`
- Новости: `data/news.json`
- Проекты: `data/projects.json`

## Локальный запуск
Самый простой способ проверить — поднять локальный http-сервер (нельзя открывать html просто двойным кликом, потому что `fetch` не работает в file://).

### Вариант 1 (Python)
```bash
python3 -m http.server 8080
```
Откройте: http://localhost:8080

### Вариант 2 (Node)
```bash
npx serve .
```

## Дальше (если хотите админку)
Можно подключить Decap CMS (Netlify CMS), чтобы редактировать `data/*.json` через страницу `/admin` и не лезть в файлы.


## Админка (без кода)

Откройте `/admin` на вашем сайте (например: `https://ВАШ-ДОМЕН/admin`).

Требуется хостинг Netlify:
1) Загрузите сайт в GitHub (или другой Git-провайдер) и подключите репозиторий в Netlify.
2) В Netlify включите **Identity** (Site settings → Identity) и затем **Git Gateway**.
3) В Identity включите регистрацию по приглашениям (Invite only) и добавьте себя как пользователя.
4) После этого зайдите на `/admin`, залогиньтесь и добавляйте товары/новости/проекты через интерфейс.

Если у вас ветка не `main`, поменяйте `branch:` в `admin/config.yml`.
